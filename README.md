Maple — Homestead Cafe AI Chatbot.

Maple is an interactive AI-powered cafe assistant designed for The Homestead Cafe. It provides a modern, conversational interface for customers to explore menus, place orders, and complete payments seamlessly.

🚀 Features--

💬 Smart Chat Interface:-
Interactive chatbot UI for customer communication
Quick action buttons for common queries
Smooth animations and typing indicators

📋 Dynamic Menu System:-
Categorized menu:
Hot Beverages
Cold Beverages
Breakfast
Mains & Snacks
Displays:
Price
Description
Tags (Veg, Popular, Spicy)

🛒 Order Management:-
Add/remove items dynamically
Quantity control system
Real-time order summary
Auto total calculation

💳 Payment Integration (UI Simulation):-
Multiple payment methods:
Card 💳
UPI 📱
Cash 💵
Secure checkout modal
Order confirmation with success screen

🎨 Modern UI/UX:-
Elegant cafe-themed design
Responsive layout (mobile-friendly)
Soft color palette (cream, brown, rust tones)
Smooth transitions and animations

🏗️ Tech Stack:-
Frontend: HTML5, CSS3, JavaScript
Fonts: Google Fonts (Playfair Display, DM Sans)
UI Design: Custom CSS with animations
Logic: Vanilla JavaScript

📂 Project Structure:-
homestead_cafe_bot.html   # Main application file
README.md                 # Project documentation

⚙️ How to Run:-
Download or clone the project
Open the file:
homestead_cafe_bot.html
Run in any modern browser (Chrome, Edge, Firefox)

🧠 How It Works:-
User interacts via chat input or quick buttons
JavaScript processes user queries
Bot responds with:
Menu details
Suggestions
Order handling
Orders are stored in a temporary state
Payment modal simulates checkout process
